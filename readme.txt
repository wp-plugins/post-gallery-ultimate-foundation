Extend the Post Gallery Ultimate plugin to use the Foundation Framework instead of the default styles.

== Description ==


Extend the Post Gallery Ultimate plugin to use the Foundation Framework instead of the default styles.

You must install the [Post Gallery Ultimate plugin](https://wordpress.org/plugins/post-gallery-ultimate/) for this plugin to work.

###What you get when you use the Post Gallery Ultimate Foundation

*  Foundation Framework essentials css and js loaded.
*  Post Gallery Ultimate template that uses Foundation Framework to display the post gallery


###Quick Start Guide

* [post gallery quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/post-gallery-ultimate-foundation/)



== Installation ==


1. Upload `post-gallery-ultimate-foundation` to the `/wp-content/post-gallery-ultimate-foundation/` directory
1. Activate the post gallery through the 'Post Gallery Ultimate Foundation' menu in WordPress
1. Click **Post Gallery** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I need the Post Gallery Ultimate plugin installed for this plugin to work? =

Yes, Post Gallery Ultimate must be installed for this plugin to work.

= Do I have to know how to program or design? =

No.  The post gallery ultimate foundation does the programming and design.
